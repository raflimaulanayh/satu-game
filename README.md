Welcome to the **Satu University Games** repository! This project is a curated collection of HTML5 web games designed to engage students, celebrate university life, and provide a fun, interactive way to explore academic themes. From chasing GPAs to battling deadlines, each game reflects a unique aspect of the student experience, wrapped in a vibrant, futuristic **Neon/Cyberpunk** aesthetic.

## üïπÔ∏è Game Library

### 1. GPA GUARDIAN (Neon Core)
**"Protect your core, protect your grades."**
*   **Genre:** Defense / Shooter
*   **Objective:** Defend the central "Core" (your GPA) from incoming "E" grades and academic obstacles. Rotate your shield and shoot to survive as long as possible.
*   **Key Features:** Minimalist neon visuals, rotary controls, increasing difficulty.

### 2. DEADLINE BREAKER (Neon Breaker)
**"Break the bricks, beat the deadline."**
*   **Genre:** Arcade / Breakout
*   **Objective:** Smash through walls of "Tasks" and "Assignments" using a ball and paddle. Clear the screen to advance to the next semester (level).
*   **Key Features:**
    *   **Campaign Mode:** Progressive levels with different layouts.
    *   **Sandbox Mode:** Customize game physics (speed, ball count, chaos mode) for unlimited fun.
    *   **Power-ups:** Multi-ball, paddle resize, and more.

### 3. SEMESTER CLIMBER (Neon Rise)
**"Rise above the challenges."**
*   **Genre:** Endless Jumper / Platformer
*   **Objective:** Guide your character (the energetic "Bee" mascot) upwards by jumping on platforms. Avoid falling into the void of academic probation!
*   **Key Features:** Vertical scrolling, precise jumping mechanics, dynamic platform generation.

### 4. SAT PT CATCHER (Neon Catch)
**"Catch the points, avoid the distractions."**
*   **Genre:** Arcade / Catcher
*   **Objective:** Move your basket left and right to catch falling "SAT Points" and "Good Grades". Avoid catching "Distractions" and "Missed Classes" that deduct score.
*   **Key Features:** Fast-paced reaction gameplay, combo system, high-score tracking.

### 5. NETWORK LINKER (Neon Void)
**"Connect the nodes, build your network."**
*   **Genre:** Puzzle / Strategy
*   **Objective:** Connect separated nodes to form a complete network. Plan your path carefully to link all academic resources without crossing lines efficiently.
*   **Key Features:** Zen-like puzzle experience, logic-based challenges.

### 6. PANDUAN MASA DEPAN (Mayor Match) üÜï
**"Temukan jurusan impianmu."**
*   **Genre:** Personality Quiz / Assessment
*   **Objective:** Answer a series of psychological and preference-based questions to discover which major at Satu University suits you best.
*   **Key Features:**
    *   **8 Deep Questions:** Designed to gauge logic, creativity, empathy, and leadership.
    *   **6 Major Outcomes:** Informatika, Sistem Informasi, DKV, Psikologi, Akuntansi, Management.
    *   **Detailed Results:** Provides a description of your personality type and why the major fits you.

## üõ†Ô∏è Technology Stack
This project is built using pure **Vanilla Web Technologies** for maximum performance and compatibility without heavy frameworks.

*   **HTML5:** Semantic structure and Canvas API for game rendering.
*   **CSS3:** Custom styling with CSS Variables, Flexbox/Grid, and Keyframe Animations for that "Neon" glow.
*   **JavaScript (ES6+):** Core game logic, physics engines, collision detection, and audio synthesis.
*   **Web Audio API:** Real-time sound generation for sound effects and dynamic background music (no external MP3s needed!).

## üöÄ How to Run
1.  **Clone the Repository:**
    ```bash
    git clone https://github.com/raflimaulanayh/satu-game.git
    ```
2.  **Open locally:**
    Simply open the `index.html` file in your favorite modern web browser (Chrome, Edge, Firefox, Safari).
3.  **No Server Required:**
    Since the games use local assets and vanilla JS, they run directly from the file system. However, for the best experience (especially for Audio API policies), running a local server (like Live Server in VS Code) is recommended.
